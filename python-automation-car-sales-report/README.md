# Automated Car Sales Report

Automates the processing of car sales data to generate summary reports
and send results via email.

## How it works

-   Reads sales data from `car_sales.json`
-   Aggregates total revenue and sales
-   Generates a text-based summary report
-   Sends the report via email

## How to run

``` bash
python3 cars.py
```
